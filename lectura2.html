CONTENIDO

HTML5 como se conoce por sus siglas en inglés (Hypertext Markuo Language, versión 5), es la quinta revisión del lenguaje HTML. En conjunto con CSS3, se define como los nuevos estándares de desarrollo web, rediseñado el código para resolver problemas y actualizando así a nuevas necesidades. No se limita solo a crear nuevas etiquetas o atributos, sino que incorpora muchas características nuevas y proporciona una plataforma de desarrollo de complejas aplicaciones web.
